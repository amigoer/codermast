# RocketMQ

Apache RocketMQ 是一个分布式消息中间件，最初由阿里巴巴开发并贡献给 Apache 基金会。具有低延迟、高可靠、高性能等特点，广泛应用于电商、金融等场景。

## 核心特性

- **低延迟**：毫秒级消息投递
- **高可靠**：消息持久化、主从复制
- **高性能**：单机十万级 TPS
- **丰富消息类型**：顺序、延迟、事务消息
